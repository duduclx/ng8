Mon premier composant

Ce composant à été créer avec la CLI

Dans le terminal:

ng generate component chapter-one

ou en raccourci

ng g c chapter-one

Le composant à été automatiquement ajouté au module racine du projet.

Dans le fichier app.module.ts, dans les imports on peut voir que le fichier chapter-one.component est importé. L'extension .ts n'est pas spécifié, car c'est le compilateur qui va ajouter cette information. et dans le tableaux des déclarations, on voit que chapterOneComponent est déclaré.

Ainsi, il devient utilisable depuis le app.module.ts, qui est le module racine de notre application.

Pourtant, si on lance la commande

ng serve

pour lancer le serveur et l'application, on voit la page par défaut, créée lors de la construction du projet avec la commande

ng new ng8

Pour afficher notre composant, on va l'intégrer dans le app.component.html

chapterOneComponent sera alors enfant (child) de appComponent

Dans chapter-one.component.ts, on voit la déclaration

selector: "app-chapter-one"

Il nous suffit d'enlever le code superflu de la page app.component.html et de le remplacer par cette balise angular.

il ne reste plus que la balise app-chapter-one et router-outlet dans la page app.component.html

il est possible de mettre en forme notre composant, via le fichier chapter-one.component.scss

attention, ce fichier scss (ou css) ne s'applique que et uniquement à ce composant. Il n'est pas possible ici de mettre en forme un autre composant.

je vais simplement formater ce chapitre à l'aide du fichier scss.
